# INFO 

## Installation

**Linux**
```
$ conda deactivate
$ python -m venv venv
$ source venv/bin/activate
$ pip install mkdocs-material
$ pip install mkdocs-awesome-pages-plugin
$ deactivate
```

**Windows**
```
> python -m venv venv
> venv\Scripts\activate.bat
> venv\Scripts\deactivate.bat
```

## Development

```
$ conda deactivate
$ source venv/bin/activate
$ mkdocs serve
```
